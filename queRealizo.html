¿Que realizo?

Páginas web :
Diseño y desarrollo sitios web estáticos y dinámicos para empresas, emprendimientos o uso personal. Trabajo con HTML, CSS, JavaScript, Bootstrap y frameworks modernos como Laravel y React.

Cable estructurado :
Instalación de redes físicas (cableado UTP, canaletas, puntos de red), organización y configuración de switches y routers para entornos empresariales o domésticos.

Páginas App :
Desarrollo de aplicaciones interactivas como sistemas de inventario, control de usuarios, gestión de ventas o automatización de procesos usando tecnologías como PHP, JavaScript, Python, APIs y bases de datos.

Desarrollo de software :
Análisis, diseño y construcción de software a medida según los requerimientos del cliente. Manejo de bases de datos, control de versiones (Git), integración de servicios externos y automatización de tareas con Python o Java.
